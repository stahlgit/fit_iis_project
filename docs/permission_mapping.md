# Permission mapping

### ADMIN

- manage users
- manage conference (create, update, delete)
- manage rooms (create, update, delete)
- approve presentations
- create time schedule

### REGISTERED USER

- submit presentation
- get rooms
- view conference schedule (get)
- book ticket
- pay for reservation
- ask question
- vote for presentation
-

### GUEST

- view conference (get)
- make reservation
